This Test Suite is from Netbeans to test the XML reader and writes due to the limitation of Android Studio.

To use please
put the XML reader/writers java classes into the XML package
put the XML files to read/write to in XML_Test_Suite.src.Accounts_Reader_Test_Files Package. Will need to modify code to accept these files

Test results are output to console.
The XML files are the test results written into the required documentation format
